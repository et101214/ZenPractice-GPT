# 禪院修行｜ZenPractice GPT

這是一款結合手抄經書、誦經陪伴、寺院場景、小沙彌生活系統與修行養成的互動式 App 原型。

## Demo V6

- Master UI 高還原介面
- 小沙彌依真實時間自動進行掃地、禮佛、抄經、打坐、用膳與睡覺
- 手寫抄經、毛筆／鋼筆／橡皮擦、字體切換與儲存
- 誦經語音、木魚節奏、速度控制與計時
- 經書選擇、翻頁與解鎖
- 小沙彌夥伴解鎖系統
- 真實時間日夜狀態與修行排程
- PWA 安裝、離線模式、連續修行天數與修行紀錄

## 線上預覽

GitHub Pages 部署完成後，可從以下網址開啟：

https://et101214.github.io/ZenPractice-GPT/

專案已加入 GitHub Actions 自動部署流程。之後只要更新 `main` 分支，就會重新部署網站。

首次使用時，請到 GitHub 專案：

`Settings → Pages → Source`

選擇：

`GitHub Actions`

## 線上檔案

`index.html` 為自包含版本，所有 UI 圖片已內嵌，可直接開啟或部署到 GitHub Pages。

## 本機預覽

Windows 可雙擊：

```text
啟動預覽.bat
```

或執行：

```bash
python dev_server.py
```

瀏覽器開啟：

```text
http://localhost:8080
```
